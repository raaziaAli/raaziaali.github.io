Linear Functions

A linear function has a constant rate of change. The slope m tells you how steep the line is. The y-intercept b is where it crosses the y-axis.

Investigation

① Drag m from negative to positive — what changes? ② Set m = 0. What kind of line do you get? ③ What does b control? ④ Find settings that make the line pass through (0, 0).

Key Idea — F-IF.6

Slope = Rise ÷ Run = (y₂−y₁)÷(x₂−x₁). A linear function in slope-intercept form is f(x) = mx + b. Domain and range are both all real numbers (unless restricted by context).

Quadratic Functions

A quadratic function creates a parabola — a symmetric U-shape with a turning point called the vertex. The rate of change is not constant.

Investigation

① What does a control? Try a = 2 vs a = 0.5 vs a = −1. ② Where is the vertex? ③ What is the axis of symmetry? ④ When does the parabola open upward vs downward?

f(x) = x²
Leading coefficient a
1
a > 0 opens up · a < 0 opens down
Horizontal shift h
0
Vertical shift k
0
Vertex: (0, 0) · opens upward · range: y ≥ 0
Key Idea — F-IF.7a

Vertex form: f(x) = a(x − h)² + k. Vertex at (h, k). If a > 0 range is y ≥ k; if a < 0 range is y ≤ k. Domain is always all real numbers. Axis of symmetry: x = h.

Exponential Functions

Exponential functions grow (or decay) at a rate proportional to their current value. The base b controls whether the function grows (b > 1) or decays (0 < b < 1).

Investigation

① Set b = 2. By how much does f(x) multiply each time x increases by 1? ② Try b = 0.5. What happens now? ③ What is the y-intercept always equal to? ④ Can an exponential function ever reach zero?

f(x) = 1 · 2ˣ
Initial value a
1
f(0) = a · b⁰ = a (the y-intercept)
Base b
2
b > 1 → growth
Table of values
Key Idea — F-LE.1

f(x) = a · bˣ where a is the initial value and b is the growth/decay factor. If b > 1: exponential growth. If 0 < b < 1: exponential decay. Domain: all reals. Range: y > 0 (never touches the x-axis — horizontal asymptote at y = 0).

Absolute Value Functions

Absolute value measures distance from zero — always non-negative. This creates the iconic V-shape. Parameters a, h, and k transform it in predictable ways.

Investigation

① Where is the vertex? ② What happens when a is negative? ③ What is the range when a < 0 vs a > 0? ④ What real-world situations create V-shaped graphs?

f(x) = |x|
Stretch / flip a
1
Negative a flips the V downward
Horizontal shift h
0
Vertical shift k
0
Vertex: (0, 0) · opens upward · range: y ≥ 0
Key Idea — F-IF.7c

Vertex form: f(x) = a|x − h| + k. Vertex at (h, k). If a > 0 range is y ≥ k (opens up). If a < 0 range is y ≤ k (opens down). Domain is always all real numbers.

A piecewise function uses different formulas for different intervals of the domain. It's like a function that changes its rule depending on where x is.

The Floor Function ⌊x⌋

The floor function returns the greatest integer less than or equal to x. Written ⌊x⌋, it always rounds down toward negative infinity — never up.

Investigation

① Why is ⌊2.9⌋ = 2 and not 3? ② Why is ⌊−0.1⌋ = −1 and not 0? ③ What is the range of the floor function? ④ Why is each step closed on the left and open on the right?

The Ceiling Function ⌈x⌉

The ceiling function returns the least integer greater than or equal to x. Written ⌈x⌉, it always rounds up toward positive infinity.

Investigation

① Compare ⌈2.1⌉ and ⌊2.1⌋. How do they differ? ② Compare ⌈3.0⌉ and ⌊3.0⌋ — are they the same? ③ On the graph, which end of each step is closed vs open — and why is it the opposite of floor?

Floor vs Ceiling Comparison

⌊2.7⌋ = 2 — rounds DOWN
⌈2.7⌉ = 3 — rounds UP
⌊3.0⌋ = 3 — integer → same
⌈3.0⌉ = 3 — integer → same
For any integer n: ⌊n⌋ = ⌈n⌉ = n
Real World — Shipping Cost 📦

C(w) = $5 · ⌈w⌉. A 2.3 lb package is charged as 3 lbs. The company rounds UP (ceiling) to ensure they never undercharge. Each step is open on the left, closed on the right.

Function Families — Summary & Matching

Every function family has a distinctive shape, equation form, and domain/range. Learn to recognise and compare all of them.

Linear

f(x) = mx + b

Straight line. Constant slope. Domain & range: all reals.

Quadratic

f(x) = a(x−h)²+k

Parabola. Vertex at (h,k). Domain: all reals. Range: y≥k or y≤k.

Exponential

f(x) = a · bˣ

Curved. Asymptote at y=0. Domain: all reals. Range: y > 0.

Absolute Value

f(x) = a|x−h|+k

V-shape. Vertex at (h,k). Domain: all reals. Range: y≥k or y≤k.

Piecewise

Multiple rules

Different formula per interval. May be continuous or have jumps.

Step · Floor · Ceiling

⌊x⌋ or ⌈x⌉

Staircase. Jumps at integers. Domain: all reals. Range: integers ℤ.
