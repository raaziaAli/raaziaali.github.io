Privacy Policy

Effective date: March 2026  ·  Last updated: March 2026

🔒 MathSpace does not collect, transmit, sell, or share any personal data. All information stays on your device.

What MathSpace does

MathSpace is a Chrome extension that adds an interactive classroom toolbar to any webpage. It provides Grade 8 mathematics teachers and students with a collaborative whiteboard, live polls, raise-hand alerts, class chat, and a teacher progress dashboard — all designed to support online learning.

Data we collect

MathSpace stores the following information locally on your device only, using Chrome's built-in chrome.storage.local API:

  • The first name entered by the student when joining a session
  • Lesson progress and XP points earned during the session
  • Whiteboard drawing strokes shared within the session
  • Chat messages and teacher hints exchanged during the session
  • Poll responses submitted during the session
  • Hand-raise status

This data is used solely to run the classroom session and display progress within the extension. It is never transmitted to any external server, database, or third party.

Data we do NOT collect

  • No names, email addresses, or account credentials
  • No browsing history or webpage content
  • No location data
  • No financial or payment information
  • No health information
  • No persistent identifiers or tracking data

How data is stored

All session data is stored in chrome.storage.local — a local storage mechanism built into Chrome that keeps data on the user's own device. This data does not leave the device. It is not synced to any cloud service, server, or third party. When a user clears their browser data or uninstalls the extension, all stored data is permanently deleted.

Third-party services

MathSpace does not use any third-party analytics, advertising, tracking, or data processing services. The extension loads a Google Fonts stylesheet for display purposes only — this is a standard web font request and does not involve any personal data.

Children's privacy

MathSpace is designed for use in Grade 8 classrooms (students approximately 13–14 years old), always under the supervision of a teacher. The extension does not knowingly collect any personal information from children. No student name, progress data, or session data is ever transmitted outside the device or accessible by anyone other than users of that same device.

Permissions used

MathSpace requests the following Chrome permissions, each used solely to deliver the classroom functionality:

  • storage — to save session state locally on the device
  • tabs — to inject the classroom toolbar into the active tab
  • activeTab — to activate the toolbar when the user clicks the extension icon
  • scripting — to inject the toolbar into pages including Google Classroom
  • alarms — to periodically sync the teacher dashboard during an active session
  • notifications — to alert the teacher when a student raises their hand

Changes to this policy

If this privacy policy is updated, the new version will be published at this URL with an updated effective date. Continued use of the extension after changes constitutes acceptance of the revised policy.
